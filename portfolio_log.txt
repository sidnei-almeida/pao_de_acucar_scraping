VISÃO GERAL
-----------
Desenvolvimento de um web scraper automatizado para coletar dados nutricionais de produtos do supermercado Pão de Açúcar. O projeto demonstra habilidades avançadas em automação web, manipulação de dados e desenvolvimento de software em Python.

---------------------

- Selenium WebDriver
- Pandas
- Logging
- Docker
- Git/GitHub

1. Web Scraping Avançado
   - Automação de navegação web
   - Manipulação de elementos dinâmicos
   - Tratamento de rate limiting
   - Gestão de sessões do navegador

2. Engenharia de Software
   - Arquitetura modular
   - Tratamento de erros
   - Logging e monitoramento
   - Versionamento de código

3. Manipulação de Dados
   - Processamento de dados estruturados
   - Exportação em múltiplos formatos (CSV, JSON)
   - Limpeza e normalização de dados
   - Tratamento de dados faltantes
